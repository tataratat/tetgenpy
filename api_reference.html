API References#

Quick overview of core classes/function#

  • tetgenpy.TetgenIO: input and output of tetgen calls are done with this object.

  • tetgenpy.PLC: helper class for creating TetgenIO. Especially helpful for avoiding explicit indexing.

  • tetgenpy.tetrahedralize: the main function of tetgen.
